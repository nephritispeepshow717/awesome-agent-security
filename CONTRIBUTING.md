# Contributing to Awesome Agent Security

Thank you for contributing! This list aims to be the most comprehensive resource for AI agent security.

## Guidelines

- **Relevance**: The resource must be directly related to AI agent identity, authorization, coordination, or security.
- **Quality**: Only include well-maintained, actively developed tools and high-quality content.
- **Format**: Follow the existing format: `[Name](URL) - Brief description.`
- **Alphabetical**: Add entries in alphabetical order within each section.
- **No duplicates**: Check that the resource isn't already listed.
- **Descriptions**: Keep descriptions concise (one sentence) and factual.

## How to contribute

1. Fork this repository
2. Add your resource to the appropriate section
3. Submit a pull request with a clear description of what you're adding

## What we're looking for

- Open source tools for agent security
- Research papers on agent authorization and identity
- Standards and protocols relevant to agent systems
- Real-world incident reports involving AI agents
- Conference talks and presentations
- Tutorials and guides

## What we don't accept

- Promotional content without substance
- Tools that are unmaintained or abandoned
- Duplicate entries
- Resources behind paywalls (unless exceptionally valuable)
